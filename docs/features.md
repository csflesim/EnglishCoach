## 產品願景 (Product Vision)

打造一個**極簡的英語反射訓練 App**,靈感來自 CIA、FSI 與情報機構的語言訓練法。

- ❌ 不是教文法。❌ 不是背單字。
- ✅ 目標:**建立自動化的英語反射**,逐步停止「母語翻譯」,直接用英文回應。

**核心理念**:`英文 → 母語 → 思考 → 英文`(傳統)→ `英文 → 直覺理解 → 英文回應`(目標)。

**目標用戶**:接近初學者的中文母語者;目標為加拿大移民、CELPIP CLB 6、IELTS。

## 核心方法 (Core Method)

每一課圍繞**一個核心句型**,透過大量高速重複把語言變成反射。**四式分開訓練**,一次只練一種模式。

### 操練類型 (Drill Types)

| 類型 | 第二層 | 狀態 | 說明 |
| :--- | :--- | :--- | :--- |
| **替換 Substitution** | 選句框 →(有變位再選人稱) | ✅ 已實作 | 句框固定,從單字庫依分類/槽位高頻替換 |
| **轉換 Transformation** | 選句框 → 選人稱 + 操作 | ✅ 已實作 | 用**變位引擎**即時生成(過去/否定/疑問/未來) |
| **擴展 Expansion** | — | ⏳ 未實作 | 規格:每次抽 5 條鏈、逐層加長(引擎與 UI 都還沒做) |
| ~~情境反應 Response~~ | — | ⏸ 關閉 | 部分單元有資料,模式關閉 |

流程:**選句型 → 選模式 →(第二層)→ 操練**。

#### 替換操練 = 漸進句框 + 單字庫變數(6 人稱)

- 句框只定義「結構 + 中文模板 + 要哪類字(category/pos/slot)」,變數由單字庫填入。
- 有變位的句框(帶 `conj`)會**輪替人稱**(I/you/he/she/we/they),或選單一人稱、或「全部輪流」;無變位句框直接替換。
- 每個句框一輪約 **20 發**(發數依間隔複習狀態遞減,見 [SRS](#間隔複習-srs)),由易到難、答錯優先。

#### 轉換操練 = 句框 × 人稱 × 操作(變位引擎)

- **兩段第二層**:先選**要變換哪個句框**,再選**人稱 + 操作**。
- 答案由**變位引擎** `renderSentence` 即時生成(非字串對應表):
  ```
  句框 I am ___. · 人稱 they · 操作 疑問
    → Are they ___?     (頂部句型同步顯示「疑問:Are they ___?」)
  ```
- 操作涵蓋:**過去式 / 否定 / 疑問 / 未來**;6 人稱皆可。

#### 文法形態自動修正(引擎層、零成本)

- **數的一致**:複數主詞 + be + 述語名詞自動變複數去冠詞(`They are a man` → `They are men`,含不規則複數)。
- **冠詞 a/an**:依後字發音自動選(`a engineer` → `an engineer`;`a university`/`an hour` 例外處理)。

## 操練流程 (Reflex Drill Flow)

```
🔊 播放提示 (cue)
  → ⏱ 計時「反應速度」(從 cue 播完到你開口那一刻)
  → 🎤 聆聽你說完整句,不打斷
  → 偵測說完 → 帶出正解,跟著複誦 → 下一發
```

- **反應速度是參考線**,不是倒數:`<1.5s` Excellent／`1.5–3s` Good／`3–6s` Learning／`>6s` Weak。反應時間記「**開口那一刻**」(非說完)。
- **一開口就讓你說完**,不中途切斷。
- **分組過場 (groupIntro)**:進新句框/新操作時先示範該句型,再連發。
- **語音不被切斷**:所有 TTS 等真正唸完 (`onend`) 才進下一步。
- **手機 TTS**:點擊手勢當下解鎖 iOS 語音。

### 三種辨識/評分模式(可並存,開關都記憶)

| 模式 | 開關 | 成本 | 行為 |
| :--- | :--- | :--- | :--- |
| **即時辨識** | 🎙 | 免費 | 瀏覽器 Web Speech 邊說邊顯示(支援的瀏覽器,如桌機 Chrome) |
| **本地比對** | ✅ | 免費 | 程式比對你說的 vs 正解,縮寫等價(I'm = I am) |
| **AI 評分** | 🤖 | gpt-4o-mini | **背景非阻塞**:練習中只收集,整輪結束一次評分 |

## AI 評分模式(背景、整輪一次)

- 🤖 開啟後,**練習中不顯示對錯、唸完正解直接下一發**(FSI 重點是反射速度,不等 AI)。
- 整輪(約 20 發)結束在**完成頁一次**送 AI(`/api/session-review`),回傳:
  - **每發對錯** + **錯誤分類**(單詞/文法/時態/冠詞/介係詞/字序/單複數/發音/用詞)
  - **整體弱點總結** + 2–4 條下一步建議
- 結果寫入 SRS(對→升 box、錯→進複習)、`error_log`(供長期分析)、drill gap。
- **FSI 教練判準**:判句型操作正確(結構/變位/人稱/語序),不要求真實慣用法(`Am I good?` 合法即正確);縮寫等價。
- **成本**:餵 Web Speech 文字給 AI(省去 Whisper 上傳);整輪一次呼叫取代每發一次。
- **未開 AI**:免費(本地比對判分或只量反應速度),不上傳音訊。

## Echo Loop(可選強化環節)

```
Target(英) → 🎙你覆述 → Native(母語) → Target(英) → 🎙你覆述 → 下一發
```

- 開啟後,帶出正解時加跑「英 → 母語錨點 → 再回英」,中間留覆述時間。
- 母語錨點來自句型庫 `nativeZh`,**非即時翻譯**;不受 🌐 翻譯開關影響。
- 預設關閉(⚙ 設定內切換)。

## 間隔複習 SRS

- **艾賓豪斯 + 萊特納 box**:`review_items`(word / sentence / drill)記 box、wrong_count、status、next_review。
- 答對升 box(間隔變長)、答錯歸零;發數依 box 遞減(20/20/10/5/3)。
- 單字 box 鏡像回 `vocabulary.box`(後台可見)。
- **drill gap**:整輪全對 → 升 box;有錯 → 歸零。
- 操練中可分別標「✗ 單詞不熟」「✗ 句子不熟」。

## 學習路徑 (Learning Path) — FSI 30 單元語言地圖

依語法骨架由淺到深排列(非主題、非 AI 隨機),分三週期:身份 → 需求 → 時態 → 條件 → 從句 → 敘事。每單元一個核心句型。

### 第一週期:點火 (Ignition) ｜ CLB 1-3
| Unit | 目的 | 核心訓練點 | 核心句型 |
| :--- | :--- | :--- | :--- |
| 1 | 身份建立 | 系動詞與身份認同 | `I am ___.` |
| 2 | 社交確認 | 一般疑問句與確認反饋 | `Are you ___?` |
| 3 | 空間定位 | 存在感與方位反射 | `There is ___.` |
| 4 | 數量感知 | 基數詞與名詞複數 | `I have two ___.` |
| 5 | 所屬關係 | 領屬代詞與財產邊界 | `This is my ___.` |
| 6 | 意志表達 | 助動詞與基本需求 | `I need ___.` |
| 7 | 職業習慣 | 一般現在時與日常行為 | `I ___ every day.` |
| 8 | 社交博弈 | 祈使句與請求 | `Could you ___?` |
| 9 | 否定防禦 | 否定結構與拒絕 | `I don't ___.` |
| 10 | 經驗回顧 | 一般過去時(規則動詞) | `Yesterday I ___ed.` |

### 第二週期:慣性 (Inertia) ｜ CLB 4-5
| Unit | 目的 | 核心訓練點 | 核心句型 |
| :--- | :--- | :--- | :--- |
| 11 | 未來投射 | 將來時與計劃 | `I will ___.` |
| 12 | 描述性感知 | 形容詞與程度修飾 | `It is very ___.` |
| 13 | 因果鏈條 | 連詞與邏輯反射 | `___ because ___.` |
| 14 | 行為頻率 | 頻度副詞與規律 | `I always ___.` |
| 15 | 動作進行 | 現在進行時與實時敘述 | `I am looking for ___.` |
| 16 | 工具使用 | 介詞與手段 | `I go by ___.` |
| 17 | 比較競爭 | 比較級與最高級 | `___ is better than ___.` |
| 18 | 義務責任 | 情態動詞 (Must / Should) | `I should ___.` |
| 19 | 許可博弈 | 情態動詞 (May / Can) | `Can I ___?` |
| 20 | 感官體驗 | 感官動詞與直接反饋 | `It looks ___.` |

### 第三週期:融合 (Fusion) ｜ CLB 6+
| Unit | 目的 | 核心訓練點 | 核心句型 |
| :--- | :--- | :--- | :--- |
| 21 | 完成狀態 | 現在完成時與結果 | `I have ___ed.` |
| 22 | 社交委婉 | 虛擬語氣與假設 | `If I were you, I would ___.` |
| 23 | 被動承受 | 被動語態與結果導向 | `It was ___ed.` |
| 24 | 複雜從句 | 定語從句與定義 | `The one that ___.` |
| 25 | 時間跨度 | 過去完成時與背景 | `I had ___ed before ___.` |
| 26 | 自我指涉 | 反身代詞與自動行為 | `I did it myself.` |
| 27 | 意向深度 | 動名詞與分詞 | `I enjoy ___ing.` |
| 28 | 社會關係 | 交互動詞與團隊 | `Let's ___ together.` |
| 29 | 決策辯論 | 複雜邏輯引導詞 | `On the other hand, ___.` |
| 30 | 敘事流 | 綜合時態轉換 | `First ___, then ___, finally ___.` |

## 掌握模型(細粒度階層)

- **句框掌握** = 該句框**每一種人稱**都練過(替換有變位:6 人稱;固定主詞:該主詞;無變位:1 次)。
- **模式掌握** = 該模式**每一個句框**都掌握。
- **單元掌握** = **每一種模式**都掌握。
- **轉換**最嚴:每句框需 **6 人稱 × 4 操作 = 24 組合**。
- 選單顯示各層進度與 ✓(模式:句框 X/Y;句框:人稱 X/6 或人稱×操作 X/24;人稱/操作逐項 ✓)。
- 首頁推薦「下一步」= 第一個尚未掌握的單元;頂端顯示「已掌握 X / 30」。

## 使用者介面 (User Interface)

底部分頁:**訓練 / 單詞 / 句子 / 分析 / 我的**。後台 `/admin` 獨立(右上 ⚙ 進入… 實為導覽列;訓練頁內另有 ⚙ 設定面板)。

- **訓練 (Training)**:主頁 = 30 單元地圖 + Next Up 推薦;選句型 → 選模式 →(第二層)→ 操練 → 完成頁(含整輪 AI 分析)。
- **單詞 / 句子 (Review)**:複習不熟/弱的單字與句子(SRS 卡片)。
- **分析 (Analysis)**:真實資料 — 弱單字/弱句子/弱句型(`review_items`)、反應太慢、常錯結構(`error_log`)、🤖 AI 深入分析。
- **我的 (Progress)**:學習地圖掌握數、連續/練習天數、**真實月曆打卡**(可翻月)。
- **⚙ 設定面板**:8 個開關(🎤/🎙/🔊/🔁/🌐/✅/🤖/🧹)+ 功能說明,全部記憶到 localStorage。

## 單字系統 (Vocabulary System)

- 由使用者匯入單字清單(住房/交通/機場/醫療/機師…),每字帶 `category` 與 `nativeZh`。
- **替換變數從單字庫依分類 + 槽位 (slot) 抓取**(不寫死);匯入越多,內容自動擴增。
- 單字另有 `pos`、`slots[]`、`difficulty`、`wordbooks[]`、`box`(已對 9393 字離線分類,含 CELPIP/IELTS 標記)。
- **使用中詞本**:選詞只從該詞本抓;後台可分頁/搜尋/選用/建立「ALL 全部」詞本。
- **AI 選詞過濾 (bad_combos)**:開始前把沒判過的「句框×字」送 AI,只排除**文法不可能**的組合,判過快取不重判,後台可還原。
- ⏳ **AI 補詞 `/api/suggest-words`**:規格保留,**未實作**(改以離線分類達成詞庫品質)。

## 弱點引擎 / 學習分析 (Weakness Engine)

> 本 App 的靈魂 —— **已由真實作答驅動**(非 mock)。

- 追蹤:答錯的字/句、反應太慢、錯誤結構分類(`error_log`)、重複犯的錯。
- 反覆失敗 → 提高複習頻率(SRS box 歸零)→ 持續出現直到掌握。
- 分析頁彙整弱點,可直接點入針對性操練。

## AI 的使用 (AI Usage)

只在必要處用 AI,皆走自家 API 路由(金鑰只在伺服器端),失敗一律退回免費模式:

1. **整輪評分** `/api/session-review`(gpt-4o-mini)— 背景模式主力。
2. **選詞過濾** `/api/check-frame`(gpt-4o-mini)— 只排除文法不可能的組合。
3. **學習分析** `/api/analyze`(gpt-4o-mini)— 分析頁深入建議。
4. **STT** `/api/stt`(whisper-1)— 後備辨識(主力為免費 Web Speech)。
5. **單發評分** `/api/evaluate`(gpt-4o-mini)— 保留(目前多由 session-review 取代)。

**不用 AI**:生成隨機課程、講解文法、產生不受控內容。課程一律來自:句型庫 + 單字庫 + 學習路徑。

## 產品原則 (Product Principle)

> 這不是英語課程、不是單字 App、不是文法 App。
> **這是一個英語反射訓練器 (English Reflex Trainer)。** 學習者重複操練句型,直到回應變得自動。
