# English Reflex Coach — 專案文件 (MVP)

一個**極簡的英語反射訓練 App**，靈感來自 CIA / FSI 的句型操練法。

> 🎯 **使命**：幫學習者建立自動化英語反射，停止「腦中翻譯」，直接用英文回應。
> 這不是英語課程、不是單字 App、不是文法 App —— 是一個**反射訓練器**。

---

## 一句話產品

打開 App → 看到一個核心句型 → 提示連發 → 你開口說 → 系統聽你說完 → 給正解與反應速度 → 重複，直到變反射。

核心方法是 FSI 四式操練(目前啟用 **替換／轉換／擴展**,情境反應暫關),四式分開、各有第二層;課程主軸是 **FSI 30 單元語言地圖**(三週期:點火→慣性→融合)。最重要的功能是**弱點引擎**(自動找出你的弱字／弱句型，反覆讓它出現直到掌握);另有可選的 **Echo Loop**(英→母語→英 錨點循環)強化記憶。**AI 評分為可選**(手動開關,回合制約 $5/月);不開則全免費(瀏覽器 TTS + 反應速度)。練完一回合可進**自由發揮模式**(自造 3 句、AI 評分)。

---

## 介面只有三頁

1. **訓練 (Training)** — 主頁即 30 單元學習地圖:Next Up 推薦下一步、各單元進度/✓,點入操練。
2. **我的弱點 (Weaknesses)** — 自動追蹤,可點選直接針對性操練。
3. **進度 (Progress)** — 練習天數、正確率、平均反應速度、估計 CLB。
