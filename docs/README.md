# English Reflex Coach — 專案文件 (MVP)

一個**極簡的英語反射訓練 App**，靈感來自 CIA / FSI 的句型操練法。

> 🎯 **使命**：幫學習者建立自動化英語反射，停止「腦中翻譯」，直接用英文回應。
> 這不是英語課程、不是單字 App、不是文法 App —— 是一個**反射訓練器**。

---

## 一句話產品

打開 App → 看到一個核心句型 → 提示連發 → 你開口說 → 系統聽你說完 → 給正解與反應速度 → 重複，直到變反射。

核心方法是 FSI 操練(目前啟用 **替換／轉換**,擴展與情境反應暫未開放),分開訓練、各有第二層;替換/轉換皆支援 **6 人稱 + 變位**(過去/否定/疑問/未來)。課程主軸是 **FSI 30 單元語言地圖**(三週期:點火→慣性→融合)。**弱點引擎**已由真實作答驅動(SRS 間隔複習 + 錯誤分類)。可選 **Echo Loop**(英→母語→英)強化記憶。評分有三模式:**🎙 即時辨識(免費)／✅ 本地比對(免費)／🤖 AI 評分(背景、整輪一次)**;不開 AI 則全免費。

---

## 介面分頁

1. **訓練 (Training)** — 主頁即 30 單元地圖:Next Up 推薦、各單元進度/✓,點入操練;完成頁含整輪 AI 分析。內含 **⚙ 設定面板**(8 開關 + 記憶)。
2. **單詞 / 句子 (Review)** — SRS 複習不熟/弱的單字與句子。
3. **分析 (Analysis)** — 真實弱點:弱字/弱句/弱句型、反應太慢、常錯結構、AI 深入分析。
4. **我的 (Progress)** — 掌握數、連續/練習天數、真實月曆打卡。
